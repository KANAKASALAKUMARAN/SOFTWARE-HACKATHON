# Full Stack Project

This is a full stack web application project.

## Prerequisites

- Node.js (Latest LTS version)

## Getting Started

1. Install Node.js from [nodejs.org](https://nodejs.org/)
2. Verify installation by running:
   ```
   node -v
   npm -v
   ```
3. Run the project:
   ```
   npm start
   ```
